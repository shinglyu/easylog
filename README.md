EasyLog
===============

A log viewer that highlights your log based on regex or smart rules.

# Install

```
./setup.sh
```

# Usage

```
./easylog <path/to/log>
```

# Testing

```
pip install -U pytest
py.test test_easylog.py
```

# Add rules

Please edit the rule in `easylog.py` for now, we'll extract that as a config file later.
